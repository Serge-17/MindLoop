# MindLoop 🔄

**MindLoop** — это интуитивно понятное и эстетичное iOS-приложение для отслеживания привычек и достижения целей. Оно помогает пользователям формировать полезные привычки, визуализировать свой прогресс и сохранять дисциплину с помощью удобного календаря и системы статистики.

## ✨ Особенности

- **Управление привычками**: Легкое создание новых целей с указанием названия, описания и целевого количества дней.
- **Визуальный прогресс**: Главный экран с индикаторами выполнения (Progress Bars) в процентах для каждой привычки.
- **Интерактивный календарь**: Детальный просмотр каждой привычки с отметками выполненных дней в календаре.
- **Аналитика**: Наглядные круговые диаграммы прогресса для оценки общей эффективности.
- **Современный UI**: Чистый, минималистичный интерфейс, созданный с использованием последних стандартов SwiftUI.

## 🛠 Технологии

- **Swift 5.9+**
- **SwiftUI** (использование круговых диаграмм, кастомных прогресс-баров и сеток)
- **SwiftData / CoreData** (предположительно, для хранения данных о привычках)
- **MVVM Architecture**

## 🚀 Установка

1. Склонируйте репозиторий:
    ```bash
    git clone https://github.com/Serge-17/MindLoop.git
    ```
2. Откройте проект в Xcode:
    ```bash
    cd MindLoop
    open MindLoop.xcodeproj
    ```
3. Соберите и запустите приложение на симуляторе iOS 17.0+.
